# .claude/skills

> 작업 맥락에 따라 자동 발동하는 스킬 모음.

## 이 폴더의 책임
- description 기반 자동 인식
- 토큰 효율적 (필요할 때만 로드)
- 사용자가 명시적 호출하지 않아도 발동

## 폴더 내용

| 스킬 | 발동 조건 | 승인 정책 (v2) |
|------|----------|----------------|
| `git-branching/` | 새 작업 시작 / 티켓 받음 | 자동 (보고만) |
| `commit-writing/` | 변경사항 발생 / 커밋 시점 | 승인 필요 |
| `pr-creating/` | 티켓 검증 완료 / PR 시점 | 승인 필요 (풀팀: 본인 머지 금지) |

## 스킬 vs 에이전트 vs 커맨드

- **스킬**: 맥락 자동 인식, description 기반
- **에이전트**: 특정 작업 위임, 명시/암묵 호출
- **커맨드**: 사용자 명시 (`/명령`)
