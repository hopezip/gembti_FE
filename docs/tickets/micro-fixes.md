# micro-fixes.md — 마이크로 변경 누적 기록

> micro 흐름으로 처리된 변경들의 한 줄 기록.
> ⚠️ **이 프로젝트는 풀팀 협업 모드라 micro 자동 흐름이 비활성화**되어 있습니다.
> 따라서 이 파일은 거의 사용되지 않으며, 오타 수정도 원칙적으로 티켓+PR 흐름을 탑니다.
> (작업 모드를 단독/소팀으로 바꾸면 다시 활성화됩니다.)

## 형식

```

```
