# docs/scratch

> 임시 메모 / 디버깅 흔적. 비공식 영역.

## 이 폴더의 책임
- 디버깅 가설/검증 메모
- 막혔을 때 임시 노트
- 학습 메모 (정식 문서화 전)

## 폴더 내용
- `<주제>-debug.md` — 디버깅 흔적
- `<주제>-notes.md` — 학습 메모

## 작업 규칙
- 비공식 — 기준 문서로 인용 금지
- 디버깅 끝나면 핵심만 티켓의 회고 섹션에 정리
- scratch 자체는 영구 보존 OK (디스크 부담되면 직접 정리)
- 다른 팀원의 scratch 임의 삭제 금지
