Design System

와이어프레임 18개 페이지에서 추출한 디자인 토큰과 컴포넌트 사양. 다크 모드 전용, 데스크탑 우선. React + Panda CSS + Park UI 스택에 그대로 적용하도록 설계되었습니다.
